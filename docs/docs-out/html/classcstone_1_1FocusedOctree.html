Perform a global update of the tree structure.

Parameters
[in]particleKeysSFC keys of local particles
[in]globalTreeLeavesglobal cornerstone leaf tree
[in]globalCountsglobal cornerstone leaf tree counts
Returns
true if the tree structure did not change

The part of the SFC that is assigned to myRank is considered as the focus area.

Preconditions:

  • The provided assignment and globalTreeLeaves are the same as what was used for calculating the list of peer ranks with findPeersMac. (not checked)
  • All local particle keys must lie within the assignment of myRank (checked) and must be sorted in ascending order (checked)
