Detailed Description

template<class T, std::size_t N>
struct util::array< T, N >

std::array-like compile-time size array

Template Parameters
Telement type
Nnumber of elements

The implementation corresponds to a device-qualified std::array minus support for length 0 plus arithmetic operations.
