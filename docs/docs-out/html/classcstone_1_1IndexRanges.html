Detailed Description

template<class Index>
class cstone::IndexRanges< Index >

Stores ranges of local particles to be sent to another rank.

Template Parameters
I32- or 64-bit signed or unsigned integer to store the indices 
                              A         B
Original particle buffer: |--—|****|-—|****|--—| Ranges: (6, 10) (14,18)

Content of offsets_: { 6, 14 } Content of scan_: { 0, 4, 8 } A B scan_ stores the offsets of a compacted buffer with the marked ranges extracted (|****|****|)
