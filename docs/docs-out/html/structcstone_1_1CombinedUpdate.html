combined update of a tree based on count-bucketsize in the focus and based on macs outside

Parameters
[in,out]treethe fully linked octree
[in]bucketSizemaximum node particle count inside the focus area
[in]focusStartstart of the focus area
[in]focusEndend of the focus area
[in]mandatoryKeysList of SFC keys that have to be present in the focus tree after this function returns. focusStart and focusEnd are always mandatory, so they don't need to be specified here. mandatoryKeys need not be sorted and can tolerate duplicates. This is used e.g. to guarantee that the assignment boundaries of peer ranks are resolved, even if the update did not converge.
[in]countsnode particle counts (including internal nodes), length = tree_.numTreeNodes()
[in]macsMAC pass/fail results for each node, length = tree_.numTreeNodes()
Returns
true if the tree structure did not change
